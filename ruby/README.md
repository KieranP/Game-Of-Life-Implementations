# Ruby

## Install

```bash
brew install ruby jruby
```

## Build

(no build step required)

## Run

### CRuby, without JIT

```bash
ruby play.rb
```

### CRuby, with YJIT

```bash
ruby --yjit play.rb
```

### CRuby, with ZJIT

```bash
ruby --zjit play.rb
```

### TruffleRuby

```bash
ruby play.rb
```

### JRuby

```bash
ruby play.rb
```
